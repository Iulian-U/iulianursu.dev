## üõ† Development

Clone the repository

```zsh
git clone https://github.com/Iulian-U/iulianursu.dev.git
```

Install dependencies

```zsh
npm install
# Or using Yarn
yarn
```

Start the development server

```zsh
npm run dev
# Or using Yarn
yarn dev
```

Build for production

```zsh
npm run build
# Or using Yarn
yarn
```
